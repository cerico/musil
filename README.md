# Clean Start
---
#### The plain and simple starter.

[![Greenkeeper badge](https://badges.greenkeeper.io/cerico/musil.svg)](https://greenkeeper.io/)

# Installation
---
Install and run: 
```
git clone https://github.com/cerico/musil
cd musil
npm install
npm run dev
```

# Write new post

```
npm run post
```

# Build 

```
npm run build
```

# Deploy

```
make deploy
```
